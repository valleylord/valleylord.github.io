物理环境和网络

理论上，Kubernetes将服务放在两台主机上。第一台主机，centos-master，是 Kubernetes 主节点，主要运行kube-apiserver, kube-controller-manager, 和 kube-scheduler，以及 etcd。另一台主机，centos-minion，是工作节点，主要运行 kubelet, proxy, cadvisor 和 docker。
